🎮 Secret Number Game (Python)

A mini game in Python where the player needs to guess a secret number with a limited number of lives. The game includes a menu system, hints, and restart functionality.

🚀 Features:
-Guess a number between 1 and 10
-Lives system
-Hint (even or odd, costs 1 life)
-Restart game
-Interactive menu loop
-User input validation

🧠 How it works:
-The program generates a random number between 1 and 10
-The player starts with 4 lives

The player can choose to:
-Try to guess
-Ask for a hint (costs 1 life)
-Restart the game
-Exit

📚 Concepts applied:
-Loops (while)
-Conditional structures (if, elif, else)
-Error handling (try/except)
-Input validation
-Flow control
-State management (lives, secret number)

🎯 Project goal:
This project was developed to practice programming logic, focusing on:
-Building interactive menus
-Controlling program flow
-Code organization

🎮 Jogo do Número Secreto (Python)

Um mini jogo em Python onde o jogador precisa adivinhar um número secreto com número limitado de vidas. O jogo possui sistema de menu, dicas e reinício.

🚀 Funcionalidades:
-Adivinhar número entre 1 e 10
-Sistema de vidas
-Dica (par ou ímpar, com custo de vida)
-Reiniciar jogo
-Menu interativo em loop
-Validação de entrada do usuário

🧠 Como funciona:
-O programa gera um número aleatório entre 1 e 10
-O jogador começa com 4 vidas

Pode escolher entre:
-Tentar adivinhar
-Pedir dica (custa 1 vida)
-Reiniciar jogo
-Sair

📚 Conceitos aplicados:
-Laços de repetição (while)
-Estruturas condicionais (if, elif, else)
-Tratamento de erros (try/except)
-Validação de entrada
-Controle de fluxo
-Manipulação de variáveis de estado (vidas, número secreto)

🎯 Objetivo do projeto:
Projeto desenvolvido para prática de lógica de programação, focando em:
-construção de menus interativos
-controle de fluxo em programas
-organização de código
